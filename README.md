Coming soon at: **taimourabid.com**

## 💻 Tech Stack

- **React 18** - UI framework
- **Vite** - Build tool & dev server
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **Calendly** - Booking integration

## 🏗️ Built With

- Modern React (Hooks, functional components)
- Responsive design (mobile-first)
- Smooth animations & gradients
- Interactive components
- Production-optimized build

## 📦 Features

- ✅ Hero section with gradient text effects
- ✅ Philosophy & approach section
- ✅ Service offerings grid
- ✅ Interactive metrics showcase
- ✅ Client logo grid (8 companies)
- ✅ Process workflow
- ✅ Embedded Calendly
- ✅ Modal popup for additional booking
- ✅ About section with co-founder links
- ✅ Full mobile responsiveness

## 🛠️ Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

## 📤 Deploy to Vercel

```bash
npm run build
```

Then connect your GitHub repo to Vercel for automatic deployments.
